#### 提醒签到app

##### 功能：

- 学习计时功能
- 自定义提醒内容
- 自定义背景图片
- 设置是否开机自启



##### 如何使用：

- 将**AlarmApp**下的压缩包解压到新建文件夹即可使用。



##### 更换背景图片：

- 建议更换大小为  **540x540的等比例** 图片



##### 软件教程：

- 打开 .exe 后缀结尾的文件，打开后左上角有设置按钮

- 设置

  - 自定义提醒内容
  - 自定义背景图片
  - 开关软件开机自启

- 主页有start按钮，点击后输入本次学习内容，即开始记录

  > 学习记录在studyrecord.txt 文件中
